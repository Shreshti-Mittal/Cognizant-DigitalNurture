# Exercise 9 - Spring Boot Application

## Objective
Create a basic Spring Boot REST application for Library Management.

## Technologies Used
- Spring Boot
- Spring Web
- Spring Data JPA
- H2 Database
- Maven

## Components
- Book Entity
- BookRepository
- BookController
- LibraryManagementApplication

## REST APIs
- GET /books
- GET /books/{id}
- POST /books
- PUT /books/{id}
- DELETE /books/{id}

## Steps Performed
1. Created Spring Boot project.
2. Added required dependencies.
3. Created Entity, Repository and Controller.
4. Configured H2 database.
5. Executed application.
6. Tested REST APIs using Postman.

## Output
Application started successfully on port 8080.
CRUD operations executed successfully.

## Result
Spring Boot REST application implemented successfully.
