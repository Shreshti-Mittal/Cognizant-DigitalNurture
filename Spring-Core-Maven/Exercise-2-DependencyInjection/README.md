# Exercise 2 - Dependency Injection

## Objective
Implement Dependency Injection using Spring IoC Container.

## Technologies Used
- Java
- Spring Core
- Maven
- XML Configuration

## Components
- BookRepository
- BookService
- applicationContext.xml

## Steps Performed
1. Created BookRepository class.
2. Created BookService class.
3. Injected BookRepository into BookService.
4. Configured bean definitions in XML.
5. Executed application.

## Output
BookService executed successfully.
BookRepository method invoked through Dependency Injection.

## Result
Setter-based Dependency Injection implemented successfully.
