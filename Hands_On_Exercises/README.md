# Hands-On Exercises

This directory contains practical hands-on exercises for the Cognizant Digital Nurture 5.0 program.

## Exercise Categories

### Java Core Fundamentals
- Basic OOP concepts
- Exception Handling
- Collections Framework

### Problem Solving
- Logic building exercises
- Algorithm implementation
- Code debugging challenges

## How to Use

1. Each exercise is placed in its own folder
2. Follow the README in each exercise folder for detailed instructions
3. Complete the tasks and test your solutions
4. Commit your solutions with clear commit messages
