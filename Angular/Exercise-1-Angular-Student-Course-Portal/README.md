\# Exercise 1: Angular Student Course Portal





\## Objective



Create a basic Angular application using Angular CLI and understand Angular project structure, components and routing.





\## Technologies Used



\- Angular

\- TypeScript

\- HTML

\- CSS

\- Angular CLI





\## Project Description



Student Course Portal is an Angular application that provides:



\- Header navigation

\- Home dashboard

\- Course listing

\- Student profile page





\## Components Created



\### Header Component



Path:



src/app/components/header



Purpose:

\- Display application title

\- Navigation between pages





\### Home Component



Path:



src/app/pages/home



Purpose:

\- Display dashboard information





\### Course List Component



Path:



src/app/pages/course-list



Purpose:

\- Display available courses





\### Student Profile Component



Path:



src/app/pages/student-profile



Purpose:

\- Display student details





\## Commands Used



```bash

ng new student-course-portal --routing --style=css



ng generate component components/header



ng generate component pages/home



ng generate component pages/course-list



ng generate component pages/student-profile



ng serve
