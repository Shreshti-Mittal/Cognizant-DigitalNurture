# Exercise 1 - Setup JUnit

## Objective

The objective of this exercise is to set up a Maven project with JUnit 5 and verify that the testing environment is working correctly.

## Concepts Covered

- Maven Project
- JUnit 5 Dependency
- Test Class
- @Test Annotation

## Files

- Calculator.java
- CalculatorTest.java
- pom.xml

## Steps Performed

1. Created a Maven project.
2. Added JUnit 5 dependency in pom.xml.
3. Created Calculator class.
4. Created CalculatorTest class.
5. Executed test cases using Maven.

mvn test
