# Exercise 4 - AAA Pattern (Arrange, Act, Assert)

## Objective

To understand the Arrange-Act-Assert testing pattern and use setup and teardown methods in JUnit.

## Concepts Covered

- @BeforeEach
- @AfterEach
- Arrange
- Act
- Assert

## Files

- Calculator.java
- CalculatorTest.java

## Steps Performed

1. Created Calculator object in @BeforeEach.
2. Executed test method.
3. Verified output using assertEquals().
4. Cleaned up after execution using @AfterEach.

```bash
mvn test
