# Exercise 4 - Employee Management System

## Scenario
This exercise implements an employee management system using arrays.

## Data Structure Used
Array

## Operations Implemented
- Add Employee
- Search Employee
- Traverse Employees
- Delete Employee

## Time Complexity
- Add: O(1), if space is available
- Search: O(n)
- Traverse: O(n)
- Delete: O(n), because shifting is required

## Array Advantages
- Simple to use
- Fast access using index
- Memory is stored contiguously

## Limitations
- Fixed size
- Insertion and deletion require shifting
- Searching is O(n) in an unsorted array
