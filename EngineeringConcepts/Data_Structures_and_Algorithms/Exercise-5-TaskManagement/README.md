# Exercise 5 - Task Management System

## Scenario
This exercise implements a task management system using a singly linked list.

## Data Structure Used
Singly Linked List

## Operations Implemented
- Add Task
- Search Task
- Traverse Tasks
- Delete Task

## Time Complexity
- Add: O(n), because we insert at the end
- Search: O(n)
- Traverse: O(n)
- Delete: O(n)

## Linked List Advantages
- Dynamic size
- Easy insertion and deletion compared to arrays
- No need for contiguous memory

## Singly Linked List
Each node contains task data and a reference to the next node.
